# initializing your app

```js
Xui.init(rootElement);
```

# mounting an element

```html
<div mount="file/path[:class]"></div> <!--no file extension-->
```

```js

export class [class name] extends XuiElement {
    constructor(el) { // el is the dom element.
        super(el); // must call
    }
}

```
